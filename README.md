# cars-counter-github

A [bouzuya/cars][] counter for GitHub.

## Installation

```bash
npm install @bouzuya/cars-counter-github
```

## Configuration

```bash
export GITHUB_USERNAME='bouzuya'
export GITHUB_ACCESS_TOKEN='...'
```

See: [`_env`](_env).
